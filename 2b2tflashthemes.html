1.20 Test update

27 March 2024

The server is updating to version 1.20 within a few days. The test server is now online, which you can join by typing '/join test' into chat while connected to the server.

The following additional changes are enabled on the test server:

  • The /kill command has been re-implemented, allowing you to die instantly when required.
  • The gamerule enderPearlsVanishOnDeath has been changed to false.

You can report any issues or bugs to [email protected] or by posting here.

Before or over this weekend, a test update will be performed forcing all players onto the test server, and it's expected that the update to 1.20 occurs before or on the weekend.

Known and possible issues:

  • Advancements not functioning
